Detailed Description

Represents a healing chamber that provides HP restoration to the player.

The heal amount is determined by the dungeon's rank with a multiplier system, and can be randomly modified by ±10%. There's also a 15% chance of receiving no healing at all.

◆ generateRandomHeal()

void ChamberHeal::generateRandomHeal ( )

Generates the final heal amount with randomization.

Calculation steps:

  1. If current heal <= 0, sets to DEFAULT_BASE_HEAL (100)
  2. Applies rank-based multiplier
  3. 15% chance to set heal to 0 (no healing)
  4. Otherwise applies ±10% random variance

Note: Double dungeon status does not affect healing amount
