Detailed Description

Represents a dungeon consisting of multiple chambers and difficulty settings.

The Dungeon class manages a dungeon's configuration, including its difficulty rank, entry requirements, combat scaling, and the chambers it contains. It tracks the player's progress, character, rewards, and dungeon features such as double dungeon status.

◆ addChamber()

bool Dungeon::addChamber ( )

Adds a chamber to the dungeon, randomly selecting between combat, heal, shop, and chest chamber. The end chamber is always boss chamber.

This method creates a new chamber of a random type (combat, heal, shop, and chest) based on the dungeon's properties and adds it to the chambers vector. It also checks if the chamber should be a double dungeon using the initRandomDoubleDungeon method. And always generate boss chamber at the last dungeon chamber

Returns
True if the chamber was successfully added, false otherwise.
