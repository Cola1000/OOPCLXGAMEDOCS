Detailed Description

Represents a single chamber in a dungeon, with difficulty rank and optional double dungeon status.

A Chamber serves as the base structure for all chamber types (boss, combat, chest, heal, and shop), carrying its difficulty rank and whether it qualifies as a double dungeon, which may alter its challenge level and rewards.

Key Attributes:

  • Rank rank: Difficulty level of the chamber.
  • bool isDoubleDungeon: Whether this chamber has double difficulty or rewards.
